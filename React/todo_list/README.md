# Todo List App

A frontend Todo List application that allows users to manage daily tasks through a clean and interactive interface.

---

## Overview

This project is a Todo List application built using **React**.  
The application allows users to add, display, and manage tasks while demonstrating component-based architecture, state management, and context usage.

The project focuses on modern React concepts such as reusable components, global state with Context API, and styled components for theming.

---

## How It Works

1. The application loads a predefined list of tasks from a data file.
2. Tasks are rendered dynamically using reusable React components.
3. The user can interact with the task list through the UI.
4. A global theme context controls the visual appearance of the application.
5. Styling is handled using global styles and theme configuration files.

---

## Features

- Display a list of todo items
- Component-based architecture
- Global theme management using Context API
- Clean and minimal user interface
- Organized project structure

---

## Technologies Used

- React
- JavaScript (ES6)
- HTML5
- CSS-in-JS (styled components)
- React Context API

## Installation

1. Download or clone the repository.
2. Navigate to the project folder:
   ```bash
   cd todo_list
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

The application will run on `http://localhost:3000`.

---

## Usage

1. Open the application in a web browser.
2. View the list of existing todo items.
3. Interact with the list through the user interface.
4. Observe how themes and global styles affect the application appearance.
