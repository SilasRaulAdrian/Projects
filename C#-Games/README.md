# C# WinForms Projects Collection

## Overview

This repository contains a **collection of C# Windows Forms projects**, including games, educational applications, and utility tools.  
All projects were developed using **C# and Windows Forms (.NET Framework 4.7.2)** and are intended for learning, practice, and demonstration of programming concepts.

The collection covers a wide range of topics such as:
- Game development fundamentals
- Event-driven programming
- Timers and animations
- Collision detection
- UI interaction
- Logic and problem-solving

---

## Technologies Used

- **Programming Language:** C#
- **Framework:** .NET Framework 4.7.2
- **UI Technology:** Windows Forms (WinForms)
- **Development Environment:** Visual Studio

## How to Run the Projects

Each project is self-contained and can be run independently.

### Option 1: Run the Executable
1. Open the project folder
2. Navigate to: bin/Debug/
3. Run the `.exe` file

### Option 2: Run from Visual Studio
1. Open the `.sln` file of the desired project
2. Build the solution
3. Press **F5** to run
