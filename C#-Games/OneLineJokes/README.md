# One Line Jokes Application

## Overview

**One Line Jokes** is a simple desktop application developed using  
**C# and Windows Forms (.NET Framework 4.7.2)**.  
The application displays short, humorous one-line jokes to the user. With a single click, a new joke is shown, providing quick and entertaining content.

This project demonstrates:
- Basic Windows Forms UI design
- Random content selection
- Event-driven programming
- Simple text display logic

---

## Application Concept

The application is designed to provide instant humor:
- A collection of one-line jokes is stored internally
- The user can generate a new joke at any time
- Each joke is displayed clearly on the screen

---

## How the Application Works

### Joke Storage

- Jokes are stored in:
  - An array, list, or predefined collection
- Each joke is a short text string

---

### Joke Generation Logic

- When the application starts:
  - A joke may be displayed automatically
- When the user clicks the button:
  - A random joke is selected from the collection
  - The displayed text is updated instantly

---

## Controls

| Action | Input |
|------|------|
| Show new joke | Button click |
| Exit application | Close window |

---

## Application Flow

1. The application launches
2. A joke is displayed on the screen
3. The user clicks the button
4. A new random joke appears
5. The process can be repeated indefinitely

---

## Visual Feedback

- Clear text display
- Simple and readable layout
- Immediate response to button clicks

---

## How to Run the Application

### Option 1: Run the Executable
1. Navigate to: OneLineJokes/bin/Debug/
2. Run `OneLineJokes.exe`

### Option 2: Run from Visual Studio
1. Open `OneLineJokes.sln`
2. Build the solution
3. Press **F5** to start the application
